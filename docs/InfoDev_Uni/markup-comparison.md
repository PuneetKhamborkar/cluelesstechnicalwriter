# Markup Language Comparison

This document compares HTML, Markdown, Asciidoc, and ReST syntax for common elements.

| Element        | HTML                | Markdown       | Asciidoc              | ReST            |
|----------------|---------------------|----------------|-----------------------|-----------------|
| Heading level 1| `<h1>Title</h1>`    | `# Title`      | `= Title =`           | `Title\n=====` |
| Bold           | `<strong>text</strong>` | `**text**` | `*text*`              | `**text**`      |
| Italic         | `<em>text</em>`     | `*text*`       | `_text_`              | `*text*`        |
| Link           | `<a href="url">link</a>` | `[link](url)`| `link:url[]`       | ``link <url>``__|
| Image          | `<img src="img">` | `![alt](img)`  | `image::img[]`        | ``.. image:: img`` |
| List item      | `<li>item</li>`     | `- item`       | `* item`              | `- item`        |

Feel free to add more rows or examples as needed.
