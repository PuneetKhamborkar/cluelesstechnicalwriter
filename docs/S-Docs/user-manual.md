## Overview

Welcome to AcmeTasker, the leading task management software designed for teams of all sizes. This document serves as the official user manual, providing comprehensive instructions on how to use the application and its features.

## Introduction

This manual covers the core functionality of AcmeTasker, helping end users understand how to use the application effectively. It includes step-by-step procedures, tips, and best practices.

## System Requirements

AcmeTasker is a web-based application compatible with the latest versions of Chrome, Firefox, Safari, and Edge. A stable internet connection is required. For mobile access, the iOS and Android apps require the latest two major OS releases.

## Installation Summary

Refer to the installation guide for setup instructions. Once the server is running, navigate to the URL provided by your administrator. The application uses SSL/TLS and should be accessed via HTTPS.

## Getting Started

After logging in for the first time, complete the profile setup wizard. Use the **+ New Task** button to create your first task, and explore the tutorial overlay for guided assistance.

## Interface Overview

The application comprises a navigation pane, task list, and detail view. Use filters to sort tasks by status, priority, or assignee. A persistent search bar is available at the top of every screen.

## Creating and Managing Tasks

Tasks contain a title, description, due date, priority, and attachments. You can assign tasks to teammates and add comments. Use subtasks and dependencies to model complex workflows.

## Projects and Boards

Group tasks into projects or boards. Each project has its own set of tasks and custom fields. Board views support kanban-style drag-and-drop task movement.

## Collaboration Features

Use mentions, comments, and file sharing to collaborate. Team members receive notifications when they are mentioned. Real-time presence indicators show who is viewing or editing a task.

## Notifications

Notifications can be configured in user settings. Choose email, SMS, or in-app alerts for task updates. Notification preferences are set per project and global defaults may be overridden.

## Settings and Preferences

Access your profile to change language, timezone, and notification preferences. Administrators can enforce password policies and manage user roles via the Admin Console.

## Keyboard Shortcuts

Press ? while in the app to display a list of available shortcuts for faster navigation. Common shortcuts include Ctrl+N for new task and Ctrl+F to focus the search bar.

## Tips and Tricks

Use tags to categorize tasks, and use the search bar to quickly locate items across projects. Pin frequently used projects and create templates for recurring task sets.

## Accessibility

AcmeTasker adheres to WCAG 2.1 AA guidelines. Screen reader support and high-contrast modes are available under the accessibility settings.

## Mobile App

The mobile app offers offline support. Changes are synchronized when the device regains connectivity. Push notifications alert you to task updates.

## Administration

Administrators can manage users, projects, and system-wide settings. Access the Admin Console via the user menu if you have the appropriate privileges.

## Backup and Restore

Data backups are performed nightly by default. Administrators may initiate manual backups and restore from previous snapshots through the Admin Console.

## Localization

The interface supports multiple languages. Users may select their preferred language in profile settings; administrators can add new translations.

## Appendix A: Glossary

- **Task**: A work item tracked in AcmeTasker.
- **Project**: A collection of related tasks.

## Appendix B: Contact Support

If you need further assistance, email support@acmetasker.com or visit our help center at https://support.acmetasker.com.
