# Installation

1. Ensure your system meets the requirements (Node.js 14+, Docker optional).
2. Download the installer package for your platform.
3. Run the installer and follow the prompts.
4. On first launch, configure your account settings.

For server deployments, unpack the archive and run `npm install` followed by `npm start`.
