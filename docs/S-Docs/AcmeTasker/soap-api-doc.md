# AcmeTasker SOAP API

The SOAP service is available at `https://api.acmetasker.com/soap`

## Operations

### GetTasks
Returns a list of tasks.

### CreateTask
Accepts task details and returns the new task ID.

### UpdateTask
Updates fields of an existing task based on ID.

### DeleteTask
Removes a task by ID.

The WSDL file is located at `/soap?wsdl`.
