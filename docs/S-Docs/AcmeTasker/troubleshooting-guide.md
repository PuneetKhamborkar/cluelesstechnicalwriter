# Troubleshooting

- **Cannot start service:** Ensure that Node.js is installed and `npm start` completes without errors.
- **API returns 401 Unauthorized:** Verify your API key is correct and included in the `Authorization` header.
- **Tasks not syncing:** Check network connectivity and server status at `status.acmetasker.com`.
- **Installation error on Windows:** Run the installer as administrator and disable antivirus during setup.

If problems persist, contact support at support@acmetasker.com.
